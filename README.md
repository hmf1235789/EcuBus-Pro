## Overview

![main](https://ecubus.oss-cn-chengdu.aliyuncs.com/img/main.png)

EcuBus-Pro is an open-source alternative to commercial automotive diagnostic tools like `CAN-OE`. It provides a comprehensive solution for ECU development and testing with:

- üÜì Open-source and free to use
- üöÄ Modern, intuitive user interface
- üíª Cross-platform support (Windows, Linux, MacOS) - [Install](./docs/about/install.md)
- üîå Multi-hardware support
  - **PEAK**: CAN, CAN-FD, LIN
  - **KVASER**: CAN, CAN-FD
  - **ZLG**: CAN, CAN-FD
  - **Toomoss**: CAN, CAN-FD, LIN
- üõ†Ô∏è Comprehensive diagnostic capabilities
  - **Diagnostic Protocols**: CAN/CAN-FD, DoIP, LIN
- üìù **Scripting**: Advanced TypeScript-based automation - [Details](./docs/um/script.md)
- üß™ **Test**: HIL Test Framework - [Details](./docs/um/test/test.md)
- üìä **Database Support**: LIN LDF (edit & export), CAN DBC (view) - [Details](./docs/um/database.md)
- üìà **Data Visualization**: Real-time signal graphing and analysis - [Details](./docs/um/graph/graph.md)
- ‚å®Ô∏è **Command Line**: Full-featured CLI for automation and integration - [Details](./docs/um/cli.md)
